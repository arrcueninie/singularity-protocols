# Books

The five volumes of the Singularity Protocols series.

Each book is available in two formats:

- **PDF** — for print, download, and offline reading
- **Markdown** — for reading directly in the browser

Both formats are identical in content. The markdown files render natively on GitHub without downloading anything.

## Suggested Reading Order

I → II → III → IV → V

Each book assumes the previous volumes. The argument repays the sequential path.

For readers who want to begin without committing to the full series, the companion documents in [`/companions`](../companions/) offer standalone entry points.
